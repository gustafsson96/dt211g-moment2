Om uppgiften

Denna uppgift utgör moment 1 i kursen Frontend-baserad webbutveckling (DT211G) hos Mittuniversitetet VT-25.

Uppgiften syftar till att:

  • Underlätta utvecklingsarbetet genom att introducera NodeJS i kombination med ett kommandobaserat verktyg för att kunna automatisera uppgifter
  • Med hjälp av Git och NodeJS skapa automatiserade utvecklingsmiljöer med automatiska publiceringar vid uppdateringar av webbplatsen
  • Förstå hur man kan skapa automatiserad transpilering och minifiering av filer i kombination med automatiserad grafikoptimering
  • Skapa en mall (så kallad "boilerplate") som kan användas som grund för kommande projekt

Uppgiften genomförs genom att skapa en responsiv webbplats med minst tre undersidor: startsida, undersida för bilder och en undersida där arbetsprocessen och automatiserad publicering presenteras.
